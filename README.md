# Cyber Threat AI Project

Professional AI-driven cybersecurity threat analysis platform.

## Setup

### Python Environment
1. The virtual environment is located in `venv/`.
2. Activate venv: `.\venv\Scripts\activate` (Windows)
3. Install dependencies: `pip install -r requirements.txt` (once created)

### Node.js Environment
1. Initialize: `npm install`
2. Start dev server: `npm start` (once configured)

## Tools Used
- Python 3.13
- Node.js 22.15
- Git
